# Session Log: Add Perl Indexer
Date: 2026-04-08

## Goal
Add `index-perl` as a new language indexer following the pattern established by the existing C, Go, PHP, Python, and TypeScript indexers. Per the user's direction, the approach was incremental — start with variable handling only, get it compiling and tested before expanding.

## Work Completed

### 1. AST Explorer (`tools/ast-explorer-perl.c`)
Created a ~90-line tool that parses a Perl file and prints the tree-sitter AST recursively. Revealed a key environment detail: the system has two tree-sitter installs:
- `/usr/lib` — v0.20.8, ABI 14 (system)
- `/usr/local/lib` — v0.26, ABI 15 (needed for the Perl grammar)

The tool must be linked with `-L/usr/local/lib -Wl,-rpath,/usr/local/lib -ltree-sitter` to avoid an ABI mismatch at runtime.

### 2. Scratch Test Files (`scratch/sources/perl/`)
Created 12 Perl source files for probing AST structure and validating indexer output:
- `variables.pl` — `my`/`our`/`local` scalars, arrays, hashes
- `subroutines.pl` — named subs, return values, prototypes
- `oop-classic.pl` — `bless`, `new`, `->` method calls
- `oop-moose.pl` — Moose `has`, `extends`, `with`, `around`
- `packages.pl` — `package` declarations, `Exporter`, `@EXPORT`
- `imports.pl` — `use`, `require`, `use constant`, `use parent`
- `regex.pl` — named regex patterns and captures
- `references.pl` — coderefs, arrayrefs, hashrefs
- `modern-perl.pl` — `use feature`, `use strict`, `say`
- `comments.pl` — inline comments and POD documentation
- `signatures.pl` — modern `sub foo($x, $y)` signatures (Perl 5.20+)
- `anonymous.pl` — anonymous subs, closures, callbacks

### 3. Config Files (`perl/config/`)
- `file_extensions.txt` — `.pl`, `.pm`, `.pod`, `.t`
- `ignore_files.txt` — `.git`, `node_modules`, `blib`, `_build`, `.build`, `inc`, `local`, `cpanm`, `.cpanm`, `fatlib`, `vendor`
- `keywords.txt` — ~60 Perl keywords

### 4. Perl Indexer Source Files
- **`perl/grammar_version.h`** — `GRAMMAR_NAME "perl"`, `GRAMMAR_VERSION "1.0.0"`
- **`perl/perl_language.h`** — `PerlParser` struct and function declarations
- **`perl/index-perl.c`** — boilerplate entry point with 4 wrapper functions, calls `indexer_main()`
- **`perl/perl_language.c`** — minimal parser (variables only):
  - Pre-looked-up `TSSymbol` IDs for `variable_declaration`, `scalar`, `array`, `hash`, `varname`
  - `index_sigil_node()` — finds `varname` child, skips single-char names (Perl internals), indexes with `CONTEXT_VARIABLE`
  - `handle_variable_declaration()` — extracts `my`/`our`/`local` modifier, walks children for sigiled nodes
  - `visit_node()` — dispatches on `variable_declaration`, otherwise recurses

### 5. `configure` Script Updates
The Makefile is generated by `configure` — all build system changes went here:
- Added `ENABLE_PERL=0` default, `--enable-perl`/`--disable-perl` flags, included in `--enable-all`
- Added `perl/grammar_version.h` generation block
- Added `#define ENABLE_PERL $ENABLE_PERL` to `config.h` generation
- Added `PERL_GRAMMAR_DIR = tree-sitter-perl` and `-I$(PERL_GRAMMAR_DIR)/src` to all 3 platform `INCLUDE_PATHS` sections (MSYS2, macOS, Linux)
- Added `PERL_TREE_SITTER_SRC/OBJ`, `PERL_LANGUAGE_SRC/OBJ`, `PERL_MAIN_SRC/OBJ` variables
- Added `$(BUILD_DIR)/index-perl` build rule and `$(PERL_GRAMMAR_DIR)/src/%.o` pattern rule
- Added `./index-perl` symlink target
- Updated `clean`, `install`, `uninstall`, `install-data` targets
- Added Perl to the configure summary output

### 6. Bug Fixes
**Compile errors in `perl/perl_language.c`:**
- `open_file()` does not exist — replaced with `safe_fopen()` + `fstat()` + `fread()` pattern (matching `python_language.c`)
- `get_relative_path()` takes 4 args (no size params) — removed the `sizeof()` arguments
- Added `#include <sys/stat.h>` for `fstat`
